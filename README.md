# My Portfolio Website

A professional portfolio website skeleton designed to showcase your skills, projects, and experience.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern Styling**: Clean, professional gradient design with smooth animations
- **Navigation**: Sticky navigation bar with smooth scrolling
- **Sections**:
  - Hero section with call-to-action button
  - About me section with image placeholder
  - Skills section with card layout
  - Projects section to showcase your work
  - Contact form for visitor inquiries
  - Footer with social links

## File Structure

```
myProfile/
├── index.html       # Main HTML file with page structure
├── styles.css       # CSS styling and responsive design
├── script.js        # JavaScript for interactions and animations
└── README.md        # This file
```

## Getting Started

1. Open `index.html` in a web browser to view the website
2. Customize the content:
   - Replace "Portfolio" in the navbar with your name
   - Update the hero section title and description
   - Add your profile image in the About section
   - Add your skills and projects
   - Update social links in the footer

## Customization Guide

### Colors
Edit the CSS variables in `styles.css` (`:root` section):
- `--primary-color`: Main brand color
- `--secondary-color`: Accent color
- `--text-color`: Text color
- `--light-bg`: Light background color

### Content
All placeholder text can be easily replaced:
- Profile image placeholder in About section
- Project cards with descriptions
- Skill categories

### Adding More Projects/Skills
Simply duplicate the card HTML and update the content. The grid will automatically adapt.
